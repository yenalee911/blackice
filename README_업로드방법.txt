## GitHub 업로드 방법
1. 이 ZIP 파일을 컴퓨터에서 압축 해제합니다.
2. GitHub 저장소 화면에서 `uploading an existing file`을 누릅니다.
3. 압축 푼 폴더 안의 파일들을 모두 드래그해서 업로드합니다.
   - index.html
   - style.css
   - script.js
   - data 폴더
4. 아래 `Commit changes` 버튼을 누릅니다.
5. 저장소 상단 `Settings` 클릭
6. 왼쪽 `Pages` 클릭
7. Branch를 `main`, 폴더를 `/root`로 설정하고 Save
8. 생성 주소 예시:
   https://yenalee911.github.io/blackice_korea/

주의: ZIP 파일 자체를 GitHub에 올리면 안 되고, 압축을 푼 안쪽 파일들을 올려야 합니다.
